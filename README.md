# leetcode-to-vscode README

Opens a temporary .py file to allow the use of VSCode extensions when writing LeetCode problems. Meant to be a dependency for this browser extension: https://github.com/umbracious/LC-to-VSC

## Features

Adds a button to LeetCode Header to open a temporary .py file in VSCode. Will add functionality for different file extensions and also make the file sync with the browser code editor.

## Requirements

https://github.com/umbracious/LC-to-VSC
